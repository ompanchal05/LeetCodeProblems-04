## 📌 About

**75 Days DSA Interview Prep (C++)** — A focused plan of **150 essential DSA problems** to build strong logic and crack 2026 tech interviews.

This repository contains my daily solutions to carefully selected Data Structures & Algorithms problems, written in C++. The goal is consistent practice, clarity of thought, and interview readiness.

## 🛠️ How to Run

Make sure you have a C++ compiler installed (e.g., `g++`).

```bash
# Clone the repository
git clone https://github.com/ompanchal05/75_DSA_Problems_solved.git
cd 75_DSA_Problems_solved

# Compile any solution
g++ -o output 1.cpp

# Run
./output
```
